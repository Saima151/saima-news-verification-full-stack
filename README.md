# saima-news-verification-full-stack
# 📰 News Verification Smart Contract (Soroban - Stellar)

## 📌 Project Description

This project is a decentralized **News Verification System** built using Soroban smart contracts on the Stellar blockchain. It allows users to submit news entries and enables verification of authenticity in a transparent and tamper-proof way.

The goal is to reduce misinformation by leveraging blockchain immutability.

---

## ⚙️ What it does

- Users can submit news with a title and content hash
- Each news entry is stored on-chain
- News can be marked as "verified"
- Anyone can fetch and check verification status

---

## 🚀 Features

- 🔐 Immutable storage on blockchain
- 🧾 News submission with unique ID
- ✅ Verification mechanism
- 👤 Tracks creator of news
- 🔎 Public access to verification status
- ⚡ Built using Soroban (Rust-based smart contracts)

---

## 🧪 How it works

1. Submit news:
   - Provide ID, title, and content hash
2. Store on blockchain:
   - Stored in contract storage
3. Verify news:
   - Mark as verified by calling verification function
4. Fetch news:
   - Retrieve and check verification status

---

## 🛠 Tech Stack

- Soroban SDK (Rust)
- Stellar Blockchain

## 📦 Functions

| Function        | Description                     |
